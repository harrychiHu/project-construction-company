第一產物保險股份有限公司 — 無障礙專區

財團法人台灣非破壞檢測協會

前端切版設計師（視覺＋網頁開發）

這是一個針對台灣非破壞檢測協會的形象官網建置專案，主要目標是提升整體網站的資訊架構與可讀性，並優化在行動裝置上的瀏覽體驗。

我負責整體視覺風格規劃、版面設計，以及 HTML / CSS 靜態頁開發。設計上著重於乾淨簡潔的閱讀性，並考量協會專業形象進行色彩配置與字體選擇。

技術上配合 RWD 響應式設計，使用 Flexbox 與 media queries 調整各裝置的排版呈現。與專案後端協作整合前後端畫面，Vue 部分與資料串接由他人負責。
