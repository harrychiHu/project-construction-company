財團法人台灣非破壞檢測協會

前端切版設計師（視覺＋網頁開發）

這是一個針對台灣非破壞檢測協會的形象官網建置專案，主要目標是提升整體網站的資訊架構與可讀性，並優化在行動裝置上的瀏覽體驗。

我負責整體視覺風格規劃、版面設計，以及 HTML / CSS 靜態頁開發。設計上著重於乾淨簡潔的閱讀性，並考量協會專業形象進行色彩配置與字體選擇。

技術上配合 RWD 響應式設計，使用 Flexbox 與 media queries 調整各裝置的排版呈現。與專案後端協作整合前後端畫面，Vue 部分與資料串接由他人負責。

德義資訊股份有限公司官網

前端切版設計師（視覺設計＋網頁開發）

みらいの夢型錄

前端切版設計師（視覺設計＋網頁開發）

這是針對みらいの夢這家日本電商平台的本店型錄網頁設計與開發專案。目標是打造一個吸引消費者目光且易於瀏覽的產品型錄頁面，以提升使用者體驗與促進銷售。

工作內容以靜態頁面切版為主，負責 HTML/CSS 實作與版面建置。設計層面著重清晰易讀與視覺一致性，並依企業形象進行配色與字體規劃。技術層面採用 RWD 響應式設計，透過 Flexbox 搭配 Media Queries，優化不同裝置尺寸下的版面呈現與使用體驗。
